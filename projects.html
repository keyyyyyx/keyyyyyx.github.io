Projects

Clustering Methods  


  • Implemented mean shift clustering and hierarchical clustering algorithms on a given dataset using R.
  • Compared their performance through visual assessment, stability analysis, and internal evaluation metrics such as within-cluster sum of squares, Silhouette score, and Calinski-Harabasz Index.
  • Gained insights into the strengths and limitations of each approach.
Tools Used: R, RStudio, tidyverse, meanShiftR, ks, Rtsne, caret, fossil.

Predicting Student Performance in Game-Based Learning  


  • Investigated the association between a student's performance on earlier questions and their performance on subsequent questions within the same gameplay session.
  • Utilizing logistic regression, random forest, and gradient boosting algorithms to predict student performance in an educational game using game logs.
  • Conducted a thorough evaluation using metrics such as accuracy and AUC, and compared the performance of each method.
Tools Used: R, RStudio, tidyverse, rigr, caret, randomForest, glmnet.

Predictive model for Perfume Prices  


  • Analyzed the key factors contributing to the retail prices of perfumes by performing exploratory data analysis (EDA) and built linear regression models using R. Used step-wise backward selection to choose the best model; assessed model performance using cross-validation.
  • Built analytical report and presentation slides using LaTex and Google Slides, clearly explained the data set and strategies used to build the model, and interpreted the final result in detail.
Tools Used: R, RStudio, tidyverse, ggplot2

Method Review of the BNT162b2 Vaccine Phase III Trial  


  • Analyzed the Bayesian workflow, specifically the prior belief and parameter transformations, proposed in the efficacy trial of BNT162b2, the Pfizer-BioNTech Covid-19 vaccine. Visualized analytical results with ggplot2.
  • Provided alternative justifiable Bayesian work-flows, highlighted with our favored philosophy of prior selection. Used grid approximation to find parameters that satisfy the proposed prior belief.
Tools Used: R, RStudio, tidyverse, ggplot2, LaTex

Databases for Animal Shelter and Hospital  


  • Designed databases for an animal shelter and a hospital using relational database design, visualized the internal relationship with entity relationship diagrams (ERD), and implemented the databases using SQL.
Tools Used: SQL

A Descriptive Model for Climate Change
